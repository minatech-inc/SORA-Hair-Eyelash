## 4. タグ設定の推奨（顧客管理用）

LINE Official Account Manager → チャット → 友だち詳細でタグを付与できます。運用が楽になる推奨タグ:

- `新規` / `リピーター` / `常連`
- `アイラッシュメイン` / `ヘアメイン` / `両方`
- `指名A` / `指名B`（スタッフ名）
- `要フォロー`（次回確認事項あり）

---

## 運用ルール（推奨）

1. **返信目安**: 営業時間内2時間以内 / 営業時間外は翌営業日
2. **定型的な質問はキーワード応答に登録** → 人の対応を減らす
3. **予約確定は必ずSquare経由で** → LINE会話だけで確定しない（ダブルブッキング防止）
4. **月次配信**: フリープラン月200通まで。新メニュー・季節キャンペーン時の一斉告知に活用
