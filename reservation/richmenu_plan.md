# LINE リッチメニュー 構成案

SORA公式LINEのリッチメニュー（トーク画面下部に常駐するボタンエリア）設計。

---

## 仕様

- **サイズ**: 2500 × 1686 px (大サイズ / 推奨)
- **形式**: PNG / JPEG
- **分割**: 6分割（3列×2行）
- **容量**: 1MB以下

## デザイン方針

- **背景**: サロンのインテリア写真 or クリーム色(#fdfbf5) ベースにシンプルに
- **アクセントカラー**: セージグリーン (#8fa085) / ウォームブラウン (#6b5641) / アイボリー
- **アイコン**: 各ボタンに細線ラインアイコン（Carousel, Calendar, Map, Blog, Instagram, Chat）
- **フォント**: 和文= Noto Serif JP (見出しのみ)、欧文= Cormorant Garamond
- **区切り線**: 白の1pxライン（6分割を視覚的に明確に）
- **全体トーン**: 上品・静謐・女性的

サイトの `style.css` のカラートークンと同じパレットを使えば一貫性が出ます。

## 作成方法

### 方法A: Canva（推奨 / 無料）

1. https://www.canva.com/ でアカウント作成
2. カスタムサイズで 2500 × 1686 px の新規デザイン作成
3. 6分割のグリッドを配置
4. 各セルにアイコン＋テキスト
5. PNG ダウンロード → LINE管理画面にアップロード

### 方法B: Claudeに作ってもらう（後日）

既存プロジェクト `MinaTech-Agent/website/scripts/` に Playwright でリッチメニューを自動生成するスクリプトがあります（[make_richmenu.py](../../MinaTech-Agent/website/scripts/make_richmenu.py)）。
SORA用にHTMLテンプレートを作成して Playwright でスクショ生成する方式も可能です。
ご希望であればClaudeに「SORA用のリッチメニューをPlaywrightで生成して」と伝えてください。

### 方法C: プロに依頼

Canvaテンプレートでも十分ですが、写真加工を含めたクオリティを求める場合は Lancers / ココナラ で 5,000〜15,000円 程度でプロに依頼可能。

## LINE管理画面での設定手順

1. **ホーム** → **リッチメニュー** → **作成**
2. テンプレートから **大(6分割)** を選択
3. 作成した画像をアップロード
4. 各セルに「リンク先URL」または「送信テキスト」を設定
5. 表示期間を設定（基本は常時表示）
6. **作成** → **適用**

---

## 保守メモ

- キャンペーン時は一時的にリッチメニューを切り替え可能（LINE管理画面で複数バージョン管理）
- 画像を差し替える際は LINE 管理画面の再アップロードが必要（URLだけの変更は不可）
- 月次で効果確認（どのボタンがよくタップされているかLINE管理画面で統計閲覧可能）
