# Square Appointments 導入手順

SORA - HAIR & EYELASH - の予約システムを Square Appointments で構築する手順書。

## 事前準備
- サロンの代表メールアドレス（Squareアカウント用）
- スタッフ全員の氏名
- 営業日・営業時間の確認（仮: 10:00-19:00 / 不定休）
- 本書類 + `square_services.csv`（全メニューチートシート）

---

## Step 1. Squareアカウント作成

1. https://squareup.com/jp/ja へアクセス
2. 「無料で始める」→ サロンの代表メールで登録
3. 業種選択: **美容・理容** → **美容室/ネイル・アイラッシュ**
4. 店舗情報入力（住所は〒253-0055 神奈川県茅ヶ崎市中海岸1-2-43 サザンマンションB棟2F）

> 💡 Square Appointments（予約管理）は **無料プラン** でスタッフ1名まで利用可能。
> 複数スタッフ対応が必要な場合は有料プラン（月額3,000円前後）。

---

## Step 2. サービス（メニュー）登録

1. Squareダッシュボード → **予約** → **サービス** → **サービスを追加**
2. [square_services.csv](./square_services.csv) を参照しながら、1件ずつ登録
3. 各サービスで設定する項目：

   | 項目 | 設定内容 |
   |---|---|
   | サービス名 | CSVの「Service Name」列 |
   | 所要時間 | CSVの「Duration (min)」列 |
   | 料金 | CSVの「Price」列（税込） |
   | カテゴリ | CSVの「Category」列 |
   | 説明文 | CSVの「Description」列 |
   | 提供スタッフ | 対応可能なスタッフにチェック |
   | バッファ時間 | 準備時間として5-10分を前後に追加推奨 |

4. **カテゴリ**を先に作成しておくと整理しやすい：
   - Eyelash（アイラッシュ）
   - Brow（アイブロウ）
   - Hair（ヘア）
   - Set Menu（セットメニュー）
   - Head Spa（ヘッドスパ）
   - Arrange（ヘアアレンジ）
   - Dressing（着付け）
   - First Time（初回限定）

> ⚠️ **所要時間は暫定値**です。スタッフで実測してから調整してください。
> バッファ時間を入れておくと、立て込んだ日でも遅延が吸収できます。

---

## Step 3. スタッフ登録

1. **予約** → **スタッフ** → **スタッフを追加**
2. 氏名・メールアドレス・担当サービス・勤務時間を設定
3. スタッフごとに予約可否を設定可能

---

## Step 4. 営業時間・定休日設定

1. **予約** → **設定** → **営業時間**
2. 営業時間: 10:00 - 19:00（要確定）
3. 最終受付: 各サービスの所要時間を考慮して設定（例: マツエク120分の場合、最終受付17:00）
4. 定休日: 「不定休」のため固定設定は不要。代わりに休みたい日を個別にブロック

---

## Step 5. オンライン予約ページ有効化

1. **予約** → **オンライン予約** → **オンライン予約を有効にする**
2. 予約ページのカスタマイズ：
   - サロン名: `SORA - HAIR & EYELASH -`
   - サロン画像: ロゴまたは店内写真
   - テーマカラー: `#8fa085`（セージグリーン / サイトと統一）
3. **予約ページURL** をコピー → これをサイトに設定

---

## Step 6. サイトに予約URLを反映

取得したSquare予約ページURLを以下に反映：

- [main.js](../main.js) の `SITE.reserve.square` を予約ページURLに書き換え
- 各HTMLファイル内の `href="#"`（Square予約ボタン）を一括置換
  - `index.html` / `menu.html` / `about.html` / `staff.html` / `access.html` / `blog.html` / `contact.html`

Claudeに「Square URL を `https://book.squareup.com/appointments/xxx` に差し替えて」と伝えれば一括対応できます。

---

## Step 7. テスト予約

1. 公開前に自分のメールアドレスで1件テスト予約
2. 確認メールが届くか、時間枠が正しく埋まるか、キャンセル可能か等を確認

---

## 連携オプション（任意）

- **Googleカレンダー連携**: 予約がGoogleカレンダーに自動同期
- **決済連携**: 事前決済・前金・当日決済を選択可能
- **SMS通知**: 予約確認・リマインダーを顧客にSMS送信（Square有料機能）
- **顧客台帳**: リピーターの履歴管理

---

## よくある注意点

| 項目 | 注意 |
|---|---|
| 初回限定メニュー | Square標準では「初回のみ適用」自動判定ができないため、別サービスとして登録し「初回の方」と明記 |
| リペア | 「前回装着日から2週間以内」の条件判定は手動確認になる |
| セットメニュー | 単品メニューと重複するが、お得感を出すため別登録推奨 |
| オプション | 単体予約できないオプション（カラーmix等）は備考欄で案内 |
